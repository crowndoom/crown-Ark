# pulse-android

xodosark’s PulseAudio Android port layer:

- `patches/`: patches applied to upstream PulseAudio (target: **v17.0**)
- `modules/`: additional Android modules (`module-*.c`) copied into the tree

Applied by `../build-android/build-pulse.sh` (and `build-pulse-android.sh`).
